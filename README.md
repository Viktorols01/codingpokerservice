# codingpokerservice
## About
This is a simple program designed to showcase interaction with codepoker.io
It currently just matches the required amount of markers.
Modify it to your own liking!

Built using **[`fastapi`](https://fastapi.tiangolo.com/)**.
## How to run

### 1. Install required packages
```bash
pip install -r requirements.txt
```
### 2. Start fastapi
```bash
fastapi dev main.py
```
